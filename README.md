## Instructions
### Navigate to Repository
```
git clone https://github.com/BekzodDjumanov/Mastermind.git
```

### Navigate to PD
```
cd Mastermind
```

### Compile
```
g++ -o Mastermind Mastermind.cpp
```

### Run
```
./Mastermind
```

**If cloning fails, please export the source code to a necessary compiler.**
